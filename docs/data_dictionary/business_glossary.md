# Enterprise Data Dictionary & Business Glossary

In order to maintain consistent reporting across the organization, all business metrics must strictly adhere to the definitions provided in this glossary.

## Core Financial Metrics

| Metric | Business Definition | Source | Data Type |
| :--- | :--- | :--- | :--- |
| **Revenue** | Total top-line sales amount *before* any discounts or taxes are applied. | ERP System | Currency |
| **Profit** | Revenue minus Cost of Goods Sold (COGS). Represents the actual money made. | ERP System | Currency |
| **Discount** | The percentage or absolute value deducted from the Revenue. | ERP System | Float |

## Machine Learning & Engineered Metrics

These metrics do not exist in the source ERP system. They are calculated dynamically by the ReadyNest Analytics Engine via `scikit-learn` and `xgboost`.

| Metric | Business Definition | Source | Data Type |
| :--- | :--- | :--- | :--- |
| **Customer Lifetime Value (CLV)** | The total historical spend of a unique customer across all their purchases. | Feature Engine | Currency |
| **Recency (Days)** | The number of days elapsed since the customer's last purchase. | Feature Engine | Integer |
| **Churn Indicator** | Binary flag (1 = Churned, 0 = Active). A customer is considered "Churned" if they have not made a purchase in 180 days. | Feature Engine | Boolean |
| **Churn Probability** | The statistical likelihood (0% to 100%) that an active customer will churn in the next quarter, determined by an XGBoost classification model. | ML Engine | Percentage |
| **Popularity Score** | A normalized 0-100 score scoring the volume of a product relative to the highest-selling product in the database. | Feature Engine | Float |

## Statistical Metrics

| Metric | Business Definition | Source | Data Type |
| :--- | :--- | :--- | :--- |
| **p-value** | Used in A/B Testing. If `p < 0.05`, the experiment (e.g. discount campaign) is statistically significant. | SciPy Engine | Float |
| **Revenue Per Visitor (RPV)** | Total Revenue generated by a group divided by the number of Visitors in that group. | SciPy Engine | Currency |
