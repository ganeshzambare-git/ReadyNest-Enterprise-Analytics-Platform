# Glossary

| Term | Definition |
|------|------------|
| **Data Lake** | The central repository of all data, encompassing raw uploads, cleaned datasets, and curated/calculated datasets. |
| **Curated Layer** | Datasets that have been processed and augmented with business logic, like Churn predictions or CLV metrics. |
| **CLV** | Customer Lifetime Value. A prediction of the net profit attributed to the entire future relationship with a customer. |
| **RBAC** | Role-Based Access Control. Securing routes and dashboards based on user roles (e.g., Admin vs Viewer). |
| **RLS** | Row-Level Security. Ensuring users only see data (rows) corresponding to their region, department, or permission level. |
| **Project Brain** | This very system. The persistent AI engineering memory layer that guides development. |
